# paste.

A beautiful, world-class text paste & save web app. Built with pure HTML, CSS, and JavaScript.

## Features
- Live line numbers
- Command Palette (Ctrl+K)
- Auto-save
- Dark & Light mode
- Copy & Download
- Context menu (right-click pastes)
- Keyboard shortcuts: Ctrl+S, Ctrl+K, Ctrl+N
- Responsive mobile support
- Persists to localStorage

## Usage
Open `index.html` in any browser.
